ArrayList

数组在创建的时候就需要指定大小，当数组的大小不够用时就需要重新申请更大的内存来存放数据，ArrayList类就被封装为一个可变长的对象数组，能够自动实现扩容。

在已有元素的末尾添加元素(size+1)

add方法首先去判断elementData的容量是否够用，如果不够用就需要对数组进行扩容，扩容后的大小为原来大小的1.5倍，当申请的1.5倍溢出的时候，就尝试以size + 1的大小申请，如果size + 1还是溢出就会抛出OutOfMemoryError()。

在指定位置添加元素 (0-size)

首先判断index在指定的[0,size]范围内，然后再进行扩容判断，随后将index以及之后的元素向后移动一位，将element元素插入到index位置上。

LockSupport

LockSupport是一个线程工具类，主要用于挂起和唤醒线程，功能上类似于Object的wait和notify。

其底层都是调用Unsafe，而Unsafe类的方法都是native方法。

线程池状态

状态 解释 ctl前三位
RUNNING 可以接受新任务并执行任务 111(此处为补码表示,原码为101)
SHUTDOWN 不接受新任务，但是会执行已经提交的任务 000
STOP 不接受新任务，同时不会执行已经提交的任务 001
TIDYING 所有的任务都已经结束，即将调用terminated() 010
